# 🛡️ Tiger Cyber Tools Portal
![Status](https://shields.io)
![License](https://shields.io)

Portal alat bantu digital (Web-based Tools) yang dirancang khusus untuk kebutuhan mahasiswa **Politeknik**, mulai dari analisis jaringan, perencanaan keuangan, hingga pemantauan kebugaran fisik.

---

## 🛠️ Main Features
Website ini mencakup berbagai modul canggih:
## 🛠️ 11 Integrated Tools

Berikut adalah daftar alat yang tersedia dalam satu dashboard:

### 🌐 Network & Security
1.  **Subnet Calculator (Pro):** Analisis IP, Network ID, Broadcast, hingga Biner.
2.  **Fake Link Analyzer:** Deteksi Phishing & Typosquatting (Anti-Googl3).
3.  **Base64 Encoder/Decoder:** Konversi teks aman (Support UTF-8 & Emoji).

### 💰 Financial & Admin
4.  **Advanced Savings Planner:** Hitung tabungan harian/mingguan untuk target jangka panjang.
5.  **Salary & Tax Analyzer:** Estimasi gaji bersih (Take Home Pay) setelah pajak/asuransi.
6.  **Percentage Calculator:** Hitung persen cepat untuk diskon, pajak, atau PPN.
7.  **Citation Generator:** Membuat sitasi format APA secara otomatis untuk tugas akhir.

### ⚙️ Technical & Engineering
8.  **Technical Unit Converter:** Konversi satuan (Inci ⇄ mm, Knot ⇄ Km/j, HP ⇄ Watt).
9.  **Ohm's Law Solver:** Hitung otomatis Tegangan (V), Arus (I), Hambatan (R), dan Daya (P).

### 🏃 Health & Productivity
10. **Cyber Fitness Planner:** Analisis BMR, TDEE, dan saran asupan kalori harian.
11. **Weight Target Finder:** Menentukan target berat badan ideal berdasarkan Rumus Broca.

---

## 🚀 Getting Started
Portal ini bersifat **Client-Side Only**, artinya tidak ada data yang dikirim ke server. Sangat ringan dan aman.
1. Clone repositori ini.
2. Buka `index.html` di browser pilihanmu.
3. Atau kunjungi versi live-nya di **GitHub Pages** kamu.
